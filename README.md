# IP_Changer with Python (ver Korean)
IP 변경 프로그램 (개발중)
<br>

## ✅구현
- 네트워크 어댑터 정보 가져오기
- 어댑터 구성 설정 변경
- 어댑터 사용, 사용안함 상태변경
- 커스텀 버튼 사전값 사용
<br>
<br>

## ❌미구현
- 운영체제 언어 확인 후 그에 맞는 정보값 저장 및 실행
- 커스텀 버튼 설정값 직접수정 및 저장
- 프리셋 TAB
- 예외처리
<br>
<br>

## ❓사용법
1. pyinstaller 사용. > pyinstaller -w -F .\main.py
2. 프로그램 관리자 권한으로 실행
